## Arithmetic Operators


```python
x = 10
y = 3
z = x + y   # addition -> 13
z = x - y   # subtraction -> 7
z = x * y   # multiplication -> 30
z = x / y   # true division -> 3.3333
z = x // y  # floor division -> 3
z = x ** y  # exponent -> 1000
z = x % y   # modulo division -> 1
```
